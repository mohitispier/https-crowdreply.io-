# CrowdReply

The #1 AI Search Visibility Platform. Track rankings, monitor cited conversations, and place your brand where it matters.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- **AI Search Tracking** - Monitor how your brand appears in AI search results
- **Conversation Monitoring** - Track every conversation where your brand is mentioned
- **Engagement Engine** - Strategically place your brand in relevant conversations
- **Ranking Analytics** - Deep insights with competitive benchmarking
- **Sentiment Analysis** - Understand how AI models perceive your brand
- **Real-time Alerts** - Instant notifications when rankings change

## Tech Stack

- Next.js 15
- React 19
- Tailwind CSS 4
- TypeScript
